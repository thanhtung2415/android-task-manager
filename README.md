# Android Task Manager

A simple Android task management app built with Kotlin and Jetpack Compose.

## Overview

This project is part of my mobile app development portfolio. The current version focuses on basic task management UI, in-memory state handling, and clean project setup with Git/GitHub.

## Features

- View sample tasks in a mobile-friendly list
- See a simple Todo / Doing / Done status summary
- Filter tasks by status: All, Todo, Doing, Done
- Add a new task with title, description, and status
- Update a task status
- Delete a task
- Material 3 UI built with Jetpack Compose

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
- Git/GitHub

## Current Status

The app currently stores tasks in memory. Data will reset when the app restarts.
